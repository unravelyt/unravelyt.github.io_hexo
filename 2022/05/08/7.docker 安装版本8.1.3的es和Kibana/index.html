Docker安装ElasticSearch和Kibana [8.1.3]

Created2022-05-08|Updated2022-06-07|3.环境搭建
|Word count:332|Reading time:1min

docker 安装ES 8.1.3

1. 安装elasticsearch

1.1 拉镜像

docker pull elasticsearch:8.1.3

docker pull kibana:8.1.3

1.2 起容器

1.3 网页登录

这里一定要是https,不然登录不了

https://192.168.32.133:9200

1.4 获取密码

docker 安装ES的 密码 和 kibana登录token 是不会显示在log上的，
所以需要手动获取密码

(1) 进入es容器内

docker exec -it elasticsearch bash

(2) 重置es登录密码

./bin/elasticsearch-reset-password -u elastic

页面输出：New value: 29qq*d_PTfOokywpM1Us

(3) 生成登录kibana token

./bin/elasticsearch-create-enrollment-token --scope kibana

得到token：
eyJ2ZXIiOiI4LjEuMyIsImFkciI6WyIxNzIuMTcuMC4zOjkyMDAiXSwiZmdyIjoiOGMzZTU5MTczMGNlZDg0N2NjODM5ZjgyNDY2OTUzZmI5Nzg3MTVkOTUwYTJlYjNjZjU0ZGI5OWM2ZDNmYjMyMyIsImtleSI6Ii1KUFJXNEFCZS1IZmFxMEpCRDdpOjNhd081VmlZU19XNDVQYThUb1phVGcifQ==

(4) 生成kibana Verification code

docker exec -it kibana bash

./bin/kibana-verification-code

2. 安装 Kibana

2.1 起容器

2.2 网页登录

这里一定要是http,不然登录不了

http://192.168.32.133:5601

登录token 在es的bin里生成的

2.3 创建新的 superUser

路径：

Stack Management > Users > create Users

2.4 kibana设置中文

docker exec -it kibana bash

echo ‘i18n.locale: “zh-CN”‘ >> config/kibana.yml
